# بلاغات الطريق - سكيكدة

منصة ويب أولية للإبلاغ عن:

- الحفر في الطريق
- تسربات المياه
- أعطال الإنارة العمومية
- انسداد المجاري أو مشاكل أخرى

## طريقة الاستخدام على GitHub Pages

1. أنشئ Repository جديد على GitHub.
2. ارفع الملفات التالية إلى جذر المستودع:
   - `index.html`
   - `style.css`
3. افتح Settings ثم Pages.
4. اختر Deploy from branch.
5. اختر branch: `main` و folder: `/root`.
6. احفظ الإعدادات.
