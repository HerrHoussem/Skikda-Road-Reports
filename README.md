# بلاغات الطريق - سكيكدة

واجهة ويب احترافية باللغة العربية لمشروع الإبلاغ عن مشاكل الطريق والخدمات العمومية في ولاية سكيكدة.

## محتويات المشروع

- `index.html` الصفحة الرئيسية الكاملة
- `style.css` ملف التصميم
- `images/skikda-banner.jpg` صورة سكيكدة المستخدمة في خلفية الصفحة الرئيسية

## طريقة الرفع إلى GitHub

1. أنشئ Repository جديد باسم `Skikda-Road-Reports`.
2. ارفع الملفات والمجلدات إلى جذر Repository:
   - `index.html`
   - `style.css`
   - `images/`
   - `README.md`
3. من Settings ثم Pages اختر:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
4. احفظ الإعدادات.

## التطوير القادم المقترح

- إنشاء صفحة `report.html`
- إضافة رفع صورة المشكلة
- إضافة GPS / تحديد الموقع
- إضافة خريطة OpenStreetMap
- ربط المشروع بـ Firebase
- إنشاء لوحة تحكم للإدارة
