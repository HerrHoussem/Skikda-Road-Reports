Services

Types de signalements pris en charge

Une interface pour faciliter la collecte et l'organisation des signalements selon le type de problème.

01

Nids-de-poule et routes endommagées

Signaler les routes détériorées pouvant représenter un risque pour les citoyens et les véhicules.

02

Fuites d'eau

Identifier les emplacements des fuites afin de réduire le gaspillage et accélérer l'intervention.

03

Éclairage public

Signaler les lampes ou poteaux défectueux pour améliorer la sécurité publique.

Trois étapes simples

Choisir le type
