# بلاغات الطريق - سكيكدة

واجهة ويب عربية جاهزة لمشروع الإبلاغ عن مشاكل الطريق والخدمات العمومية في ولاية سكيكدة.

## الملفات

- `index.html` الصفحة الرئيسية بعد التصحيح
- `style.css` التصميم الكامل بعد إصلاح مشكلة الكلمات المتلاصقة
- `images/skikda-banner.jpg` صورة المدينة كخلفية رئيسية

## ملاحظات مهمة

- ارفع `index.html` و `style.css` ومجلد `images` مباشرة داخل جذر Repository.
- لا تضع الملفات داخل مجلد داخلي إضافي إذا كنت تريد GitHub Pages يقرأ الصفحة مباشرة.
